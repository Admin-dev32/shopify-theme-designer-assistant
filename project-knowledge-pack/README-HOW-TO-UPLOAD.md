# How to Upload This Knowledge Pack

This folder contains a small Project Knowledge pack for a ChatGPT Project named:

```text
Shopify Theme Designer Assistant
```

## Upload steps

1. Create a new ChatGPT Project named `Shopify Theme Designer Assistant`.
2. Upload all files from `project-knowledge-pack/` into the Project Knowledge area, including `10-shopify-square-csv-upload-rules.md`.
3. Open `01-project-instructions.md`.
4. Copy the full contents of `01-project-instructions.md` into the Project Instructions field for the ChatGPT Project.
5. Use this ChatGPT Project whenever working on Shopify design, Shopify implementation specs, Shopify Liquid/HTML/CSS/JS code, Shopify editor debugging, or Codex prompts for real Shopify theme repos.

## What this pack is

This pack is intentionally small and focused. It gives the ChatGPT Project practical rules for acting as a Shopify expert assistant.

It is for:

- Planning Shopify sections and headers.
- Turning visual ideas into implementation specs.
- Generating Shopify-ready code when requested.
- Debugging Shopify theme editor issues.
- Creating high-quality Codex prompts for real theme repositories.

## What this pack is not

This pack is not:

- A Shopify theme.
- A Codex runtime repository.
- A place to store final storefront code.
- A large documentation mirror.

## Future additions

Later, concise summaries of official Shopify documentation can be added if needed. Keep the pack small and practical so the Project remains easy to use.
